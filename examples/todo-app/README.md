# Getting Started with Todo App

A Todo app example that does the following:
* State management using valtio and session-store.
* Valtio is used for managing local state while session-store is used for persisting data in session storage.

## Available Scripts

In the project directory, you can run:

### `npm start`

Runs the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in your browser.
